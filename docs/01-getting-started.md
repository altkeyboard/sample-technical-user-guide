# Getting Started

## Prerequisites

Before accessing SecureGrid Monitor, ensure you have:

- An active internal network account
- Authorized system access permissions
- A supported web browser (Chrome, Edge, or Firefox)

If you do not have access, contact your system administrator.

---

## Accessing the System

1. Open your supported web browser.
2. Navigate to the SecureGrid Monitor internal URL.
3. Enter your assigned username and password.
4. Select **Log In**.

Upon successful login, you will be directed to the Dashboard.

---

## Dashboard Overview

The Dashboard displays:

- Active alerts
- System status indicators
- Recent activity logs
- Notification summaries

Users can navigate to detailed views using the top navigation bar.

---

## First-Time Setup

After logging in for the first time:

1. Select your profile icon in the upper-right corner.
2. Open **Settings**.
3. Configure notification preferences.
4. Confirm time zone settings.
5. Select **Save Changes**.

Proper configuration ensures timely alert notifications.
