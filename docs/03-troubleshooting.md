# Troubleshooting

This section outlines common issues and recommended resolutions when using SecureGrid Monitor.

---

## Login Failure

**Issue:**  
User receives an "Invalid Credentials" error.

**Possible Causes:**
- Incorrect username or password
- Expired credentials
- Account lockout due to multiple failed attempts

**Resolution:**
1. Verify username spelling.
2. Reset password using the internal password management system.
3. If locked out, contact your system administrator.

---

## Alerts Not Displaying

**Issue:**  
No alerts appear on the Dashboard.

**Possible Causes:**
- Active filters applied
- Network connectivity issue
- System maintenance window

**Resolution:**
1. Select **Reset Filters**.
2. Refresh the browser.
3. Confirm internal network connectivity.
4. Verify system status with IT support.

---

## Report Generation Failure

**Issue:**  
Report fails to download after selecting "Generate Report."

**Possible Causes:**
- Unsupported browser
- Pop-up blocker enabled
- Large data range selected

**Resolution:**
1. Ensure you are using a supported browser.
2. Disable pop-up blockers.
3. Reduce the report date range and try again.

---

## Notification Settings Not Saving

**Issue:**  
Changes to notification settings do not persist.

**Possible Causes:**
- Session timeout
- Browser cache issue

**Resolution:**
1. Log out and log back in.
2. Clear browser cache.
3. Reapply notification settings and select **Save Changes**.

---

If issues persist after following these steps, escalate to the IT support team with relevant error details and timestamps.
