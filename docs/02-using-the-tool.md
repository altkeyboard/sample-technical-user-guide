# Using the Tool

## Viewing Active Alerts

The Alerts panel displays all active system alerts in real time.

To view active alerts:

1. Select **Alerts** from the top navigation bar.
2. Review the alert list displayed in the main panel.
3. Select an alert ID to open detailed information.

Alert details include severity level, timestamp, affected system, and recommended action.

---

## Filtering Alerts

Users can filter alerts to narrow results.

To apply a filter:

1. Select the **Filter** button above the alert list.
2. Choose filter criteria (severity, date range, system type).
3. Select **Apply Filters**.

To clear filters, select **Reset Filters**.

---

## Exporting Reports

SecureGrid Monitor allows users to export alert data.

To export a report:

1. Navigate to the **Reports** tab.
2. Select a report type.
3. Define the date range.
4. Select **Generate Report**.
5. Choose **Download CSV** or **Download PDF**.

Exported reports can be shared with authorized stakeholders.

---

## Managing Notification Settings

Users can customize alert notifications.

1. Select your profile icon.
2. Open **Settings**.
3. Select **Notifications**.
4. Choose delivery method (Email or SMS).
5. Select alert severity preferences.
6. Select **Save Changes**.

Proper notification configuration ensures timely response to system events.
