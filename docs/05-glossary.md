# Glossary

## Alert
A system-generated notification indicating a detected anomaly or operational condition requiring review.

## Alert Severity
The classification level assigned to an alert (e.g., Low, Medium, High, Critical).

## Dashboard
The main landing page displaying system status, alerts, and activity summaries.

## Filter
A tool used to refine displayed results based on selected criteria.

## Incident
A confirmed operational issue requiring corrective action.

## Log
A time-stamped record of system activity.

## Notification
A message delivered to users via email or SMS to inform them of system events.

## Report
A generated summary of alert data within a specified time range.

## Session Timeout
Automatic logout after a period of inactivity.

## System Administrator
An authorized user responsible for managing access, permissions, and configuration settings.

## User Role
The level of system permissions assigned to a user.
